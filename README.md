# Travel Companion

Travel Companion is a web app that serves you the basic details about your next trip's destination : )

## Features

- An overview on your destination
- Best restaurants
- Next 3 days' weather forecast
- Location on the map and its surroundings
- Latest news and information



## Tech Stack

**Client:** ReactJs

_Additional NPM packages_

- axios
- mapbox-gl
- react-toastify
- react-router-dom

**Server:** Node, Express

_Additional NPM packages_

- cors
- dotenv
- nodemon
- bcryptjs
- mongoose
- concurrently

**Database:** MongoDB



## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`MONGO_URI`

`JWT_SECRET`

## To Run Locally

Clone the project

```bash
  
  npm install
```

For client side

```bash
  cd client
  npm install
```

Start

```bash
  npm run dev
```
